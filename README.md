
# 📊 E-Commerce Exploratory Data Analysis (EDA) Dashboard

## 📌 Project Overview

This project demonstrates Exploratory Data Analysis (EDA) on an E-Commerce dataset using Microsoft Excel. The goal was to analyze sales data, identify business trends, and build an interactive dashboard.

---

## 🎯 Objectives

- Clean and prepare the dataset
- Perform descriptive statistical analysis
- Analyze customer purchasing behavior
- Create Pivot Tables and Pivot Charts
- Design an interactive Excel dashboard
- Extract meaningful business insights

---

## 🛠 Tools Used

- Microsoft Excel 2016
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization

---

## 📈 Dashboard Analysis

The dashboard includes:

- Orders by Product
- Orders by Payment Method
- Order Status Distribution
- Orders by Referral Source
- Revenue by Product

---

## 📊 Key Insights

- Total Orders: **1200**
- Most Ordered Product: **Printer**
- Most Used Payment Method: **Online**
- Most Common Order Status: **Shipped**
- Top Referral Source: **Instagram**
- Highest Revenue Product: **Printer**

---

## 📂 Dataset

The dataset contains:

- OrderID
- Date
- CustomerID
- Product
- Quantity
- UnitPrice
- PaymentMethod
- OrderStatus
- ReferralSource
- TotalPrice
